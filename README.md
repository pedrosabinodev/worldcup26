# World Cup 26 — Do You Know The Players?

A football trivia game for the FIFA World Cup 2026. Guess which country each player represents, then see their photo and club revealed.

**Live at:** [your-vercel-url.vercel.app]

## Built with
- Plain HTML + CSS + Vanilla JavaScript (no framework)
- Wikipedia API for player photos
- Vercel for hosting

## Features
- 3 difficulty levels (Stars Only / Mixed / All Players)
- 56 players from confirmed WC2026 nations
- Card flip photo reveal
- Streak scoring system
- Club badge reveal
- Google AdSense ready

## Run locally
Just open `index.html` in your browser. No install needed.

## Deploy
Push to GitHub → connect to Vercel → done.
